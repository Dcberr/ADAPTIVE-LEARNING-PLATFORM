# Review API

## 1. Review Submission

### Endpoint

POST /reviews/submission/{submissionId}

---

### Description

Run AI-based code review for a submission

---

### Path Params

| Param        | Type | Description   |
| ------------ | ---- | ------------- |
| submissionId | UUID | Submission ID |

---

### Response

```json
{
  "success": true,
  "data": {
    "id": "uuid",
    "feedback": "Code is correct but can be optimized...",
    "score": 8,
    "createdAt": "2026-03-22T10:00:00Z"
  }
}
```

---

## 2. Get Review History

### Endpoint

GET /reviews/submission/{submissionId}

---

### Description

Get all review results of a submission

---

### Response

```json
{
  "success": true,
  "data": [
    {
      "id": "uuid",
      "feedback": "Good solution",
      "score": 9
    },
    {
      "id": "uuid",
      "feedback": "Needs improvement",
      "score": 6
    }
  ]
}
```

---

## Authentication

* Required for all endpoints

---

## Notes

* Review is generated by AI service
* Multiple reviews can exist for one submission
